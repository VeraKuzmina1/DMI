    • mkdir - izveidot katalogu
    • mv - pārvietot vai pārdēvēt failu
    • nano - teksta redaktors
    • rm - dzest failu vai mape
    • rmdir - dzest katalogu
    • pwd - tekoša kayaloga nosaukums
    • cd - pārvietošanas operants
  • shell: darbības ar git'u
    • git clone - lejupielādēt viss, kas ir githabā

    • git commit -m -
    • git push origin master - augšupielāde github
    • git pull - lejpielādēt nodzēsto repozitāriju
